# Automation

Personal repository for automation scripts used in marketing, ecommerce and data workflows.

## Structure

python  
Local Python scripts (image processing, scraping, data tools)

google_app_scripts  
Scripts used in Google Sheets and Google Ads

---

## Open project folder in Terminal

Navigate to the automation project:

```bash
cd ~/dev/automation
```

Check current location:

```bash
pwd
```

---

## Basic Git workflow

Save and push changes to GitHub:

```bash
git add .
git commit -m "update"
git push
```

---

## Sync changes on another computer

Download the latest version:

```bash
git pull
```
